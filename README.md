# A-Share Quantitative Strategies

> 双策略 A 股量化交易系统 | 300,000 RMB 实盘运行中

---

## 策略

### 🛡️ 5-Gate V3（主力）

五层硬性筛选 + 三级市场自适应。对单只候选标的做"买入/观望/拒绝"判断。

| Gate | 过滤 | 规则 |
|:-----|:-----|:-----|
| 1-涨幅 | 30 日涨幅 | 🟢牛<50% / 🟡震<20% / 🔴熊<20% |
| 2-趋势 | 均线 | 价>MA20 且 MA60 不拐头 |
| 3-估值 | PE | 科技<200 / 传统<100 |
| 4-量比 | 成交量 | 🟢牛>0.7 / 🟡震>0.85 |
| 5-消息 | 负面 | 实盘必查 |

**三级市场体制：**

| 市场 | 条件 | 仓位 | 止损 |
|:-----|:-----|:---:|:---:|
| 🟢 牛市 | SH > MA20 | 25%/只 | -5% |
| 🟡 震荡 | MA50 < SH < MA20 | 12.5%/只 | -5% |
| 🔴 熊市 | SH < MA50 | 空仓 | — |

**六因子评分（110 分满分）：**
动量 20 + 量比 20 + 趋势 20 + 涨幅适中 15 + PE 15 + 科技 +10

---

### 🔥 烽火 V5（辅助）

全市场批量筛选，五因子评分 + 两级大盘过滤。

| 因子 | 权重 | 说明 |
|:-----|:---:|:-----|
| 动量 | 25% | 5 日涨幅 |
| 流动 | 25% | 换手率 |
| 趋势 | 20% | 均线排列 |
| 板块 | 15% | 行业热度 |
| 启动 | 15% | 突破质量 |

## 脚本

| 脚本 | 功能 |
|:-----|:-----|
| `intraday_stop.py` | 盘中监控（止损+补仓+熔断+跌停保护） |

---

## 风控体系

| 保护 | 触发条件 | 动作 |
|:-----|:-----|:-----|
| 止损 | 价 ≤ 止损线 | 卖出 |
| 跌停保护 | 开盘低开 > 8% | 立即以开盘价卖出 |
| 熔断 | 连续 3 次止损 | 清仓 + 暂停 5 天 |
| T+1 合规 | 当日买入 | 锁定至次日 |
| 仓位上限 | 每策略最多 4 只 | 补仓时自动检查 |
